# ディレクトリについて

bag_file内にrosbagなどの実験dataを入れてください。おいおいいい感じにdriveに自動的にgoogle driveに上げてくれるサムシングを作ります

build, log, install, .venvはignoreしています。
.venvについては[こちら](https://zenn.dev/kimushun1101/articles/ros2-jazzy-pip)を参照してください。
humbleだとあまり気にしなくて良いですが、以降を見ればわかるように基本的にpipや、sudo make installで端末の環境をいじらないように気をつけています。
みなさんも安易に依存するツールを端末の環境を改変して入れることがないように注意してください。
